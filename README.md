## Como Executar

1. Certifique-se de ter o Python instalado (recomenda-se Python 3.8+).
2. Instale as dependências necessárias:
   ```bash
   pip install numpy
   ```
3. Execute os exemplos desejados:
   ```bash
   python LinearRegression/example1.py

   ```
